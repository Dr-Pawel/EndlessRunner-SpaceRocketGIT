# Space Rocket

**Simple endless runner type game made in Unity Engine.**    
**Avoid asteroids, collect points and unlock new vehicles.**

**Features include:**    
* 3 achievements to unlock*    
* point shop with 3 spaceship skins*    
* save reset function!*
